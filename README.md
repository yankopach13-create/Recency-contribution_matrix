# Recency Contribution Matrix

Streamlit-приложение для анализа вклада клиентов по реценси (месяц последней покупки).

## Описание

- **База (папка `base/`):** Excel-файлы с покупками по датам и кодам клиентов. Колонки: Группа1, Дата, Клиентов, Код клиента. Название файла — категория и период (например 2024, 2025).
- **Режим «только база»:** сканирование base → для каждого клиента определяется месяц последней покупки → круговая диаграмма: доля клиентов или доля по полю «Клиентов» по месяцам реценси.
- **Режим «база + загрузка»:** пользователь загружает файл с кодами клиентов, выручкой и/или продажами в шт. → джойн с базой по реценси → круговая по выручке или штукам по месяцам последней покупки.

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой на Streamlit Cloud

Подключить репозиторий к [Streamlit Cloud](https://share.streamlit.io), главный файл — `app.py`.
